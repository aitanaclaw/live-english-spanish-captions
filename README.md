# Live English → Spanish Captions

Web app gratuita para iPhone/Safari que escucha inglés con el micrófono y muestra traducción al español en vivo.

- Subtítulos nuevos arriba
- Altura máxima para no empujar la pantalla
- Cierre automático de oración cuando hay pausa
- Historial opcional

> Usa APIs del navegador para reconocimiento de voz y Google Translate público para traducción.
